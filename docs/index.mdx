## My super album app

This app allows you to keep a database of existing albums in your collection

### Example set up

1. Clone the repo
1. Add your api key.

   **Example:**

   ```
   API_SECRET="wJalrXUtnFEMI#K7MDENG/bPxRfiCY3X1MP12K3d"
   ```

1. Run the application.
